# Задача 3 — Матричное уравнение токов

## Условие

Заданы матрица сопротивлений:

$$
R_{i,j} = \begin{cases}
\cos(j(i+j)), & i = j \\
1, & i = j+1 \\
0, & \text{иначе}
\end{cases}
$$

и вектор напряжений:

$$
U_i = i
$$

Найти токи $I$ из матричного уравнения:

$$
RI = U
$$

для $i = 5, 6, \ldots, 80$, $j = 5, 6, \ldots, 80$.

## Решение

Система решается методом `numpy.linalg.solve`.

## Запуск

```bash
python solution.py
```
